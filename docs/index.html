faq

Common questions.

Is this a hack or bypass for Roblox?

No. SpoofTrap doesn't modify Roblox or the account. It wraps the usual launch path with a local DPI-bypass layer so restrictive networks don't block the connection.

Why does it need DPI bypass?

Schools, dorms, and corporate networks often inspect TLS traffic and block Roblox at the handshake. SpoofTrap ships a bundled spoofdpi binary to fragment and shift the TLS so the handshake completes.

Is it signed? Will Gatekeeper block it?

Yes — DMG, PKG, and ZIP are code-signed and notarized by Apple. If macOS still warns on first launch, right-click and choose Open.

What does Pro unlock?

Higher FPS caps (144, 240, unlimited), multi-instance, custom FastFlag entries, and extra presets. Keys are issued by hand in the Discord — no subscription, no checkout. Full pricing →

Does it collect any data?

No telemetry, no analytics. Pro license validation hits Supabase; everything else runs locally. The source is on GitHub.

What about Windows?

Windows is in public beta. Rough around the edges, provided as-is. Read the warning before installing. Windows page →
